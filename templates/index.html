Bienvenue sur l'aide à la recherche de l'annuaire d'entreprise !

Pour trouver des personnes dans l'annuaire, vous pouvez utiliser des filtres et des jokers pour affiner votre recherche. Voici quelques exemples pour vous aider :

Recherche par nom :

  • Utilisez * pour afficher toutes les personnes de l'entreprise.

  • Utilisez mot* pour afficher toutes les personnes dont le prénom commence par "mot".

  • Utilisez *mot pour afficher toutes les personnes dont le nom se termine par "mot".

  • Utilisez mot * pour afficher toutes les personnes dont le prénom est "mot".

  • Utilisez mot* f pour afficher toutes les personnes dont le prénom commence par "mot" et le nom commence par "f".

  • Utilisez a pour afficher toutes les personnes dont le nom ou le prénom contient la lettre "a".

  • Utilisez col pour afficher toutes les personnes dont le nom ou le prénom contient "col".

Recherche par date de naissance :

  • Utilisez >1999 pour afficher toutes les personnes nées après 1999.

  • Utilisez <1999 pour afficher toutes les personnes nées avant 1999.

  • Utilisez 1980-1999 pour afficher toutes les personnes nées entre 1980 et 1999.

  • Utilisez 10/05/1999 pour afficher toutes les personnes nées le 10 mai 1999.

  • Utilisez 1999 pour afficher toutes les personnes nées en 1999.

Filtrage :

  • Utilisez (Présidence) * pour afficher toutes les personnes faisant partie du groupe "Présidence".

  • Utilisez (Présidence, Communication, Ressources Humaines) col pour afficher toutes les personnes dont le nom ou le prénom contient "col" et qui font partie de l'un des groupes suivants : "Présidence", "Communication" ou "Ressources Humaines".

N'oubliez pas que vous pouvez combiner plusieurs filtres et jokers pour affiner votre recherche encore plus. Par exemple, vous pouvez utiliser (Présidence) mot* pour afficher toutes les personnes dont le prénom commence par "mot" et qui font partie du groupe "Présidence".
